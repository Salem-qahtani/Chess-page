# Chess Landing Page

A chess-themed landing page built to practice HTML and CSS fundamentals.

## Live Preview

[View on GitHub Pages](https://salem-qahtani.github.io/Chess-page/)

## About

This project is a static landing page designed to introduce users to the game of chess. It features a hero section, a showcase of top chess players, iconic chess quotes, and a call-to-action to join chess.com.

## Sections

- **Header** — Navigation bar with logo and links
- **Hero** — Welcome message with a join button
- **Top Players** — Cards for Magnus Carlsen, Bobby Fischer, Hikaru Nakamura, and Mikhail Tal
- **Quotes** — Famous quotes from chess legends
- **Call to Action** — Prompt to join chess.com
- **Footer** — Copyright

## Built With

- HTML
- CSS (Flexbox)
- [Font Awesome](https://fontawesome.com/) — chess knight icon
- [Google Fonts](https://fonts.google.com/) — Instrument Serif, Playfair Display

## What I Learned

- Built the entire page from scratch without templates or frameworks
- Flexbox layout and alignment
- Responsive sizing with percentages
- CSS hover effects
- Linking external fonts and icon libraries
- Git version control and deploying with GitHub Pages
